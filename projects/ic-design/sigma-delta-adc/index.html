Sigma Delta Modulator ADC

Project Overview

This project involved the simulation and optimization of noise-shaping architectures for a Sigma Delta Modulator (SDM) Analog-to-Digital Converter (ADC). The primary goal was to achieve high-precision conversion suitable for modern sensor interfaces. The entire design and simulation process was conducted using the Cadence Virtuoso suite.

Key Features & Technologies

  • Technology: Cadence Virtuoso
  • Core Concept: Noise Shaping and Oversampling
  • Application: High-precision data conversion
