Level 2 Autonomous Car

Project Overview

This project is my final Electrical Engineering bachelor thesis, which received the Best Research award. It is a Level 2 Advanced Driver-Assistance System (ADAS) based on SAE standards, designed to be uniquely compatible with older cars featuring semi-drive-by-wire systems. The core innovation is its plug-and-play nature, allowing modern autonomous features to be retrofitted into vehicles that typically do not support them.

The system was successfully tested by 109 people during a three-day public exhibition and was also personally tested by the President of Software from Toyota, Japan. This project was proudly funded by LPDP and Riset.AI Ltd.

Key Features & Technologies

  • Automatic Emergency Braking (AEB)
  • Adaptive Cruise Control (ACC)
  • Lane Keeping Assist (LKA)
  • Driver Monitoring System
  • Vehicle: Implemented on a Honda Brio.
